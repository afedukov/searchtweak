# Team Management

Team Management controls access, collaboration, and operational governance.

## Team Roles

- **Owner**
  - Full control (same practical capability as admin + ownership privileges)
- **Administrator**
  - Can manage endpoints, models, evaluations, feedback, judges, and team settings
- **Search Evaluator**
  - Focused permissions for grading and leaderboard visibility

## What Admins Usually Do

- invite/remove members
- assign/change roles
- manage tags
- manage API token
- monitor evaluation and judge operations

## Member Operations

- invite by email
- track pending invitations
- remove users from team
- (optional) send team messages

## Team-Scoped Data Model

All core entities are team-scoped:

- search endpoints
- search models
- search evaluations
- judges
- tags

Meaning: cross-team access is isolated by design.
