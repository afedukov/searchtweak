# Leaderboard

Leaderboard shows grading throughput for both humans and AI judges.

## Modes

- `All`: users + judges in one ranking
- `Users`: human-only ranking
- `Judges`: AI-only ranking

The selected mode affects both:

- table entries
- chart series

## Filters

- Date range
- Tag filters
- Mode (`All | Users | Judges`)

## Reading Leaderboard Correctly

- Compare same date range when analyzing productivity changes.
- Use `All` for operations overview.
- Use split modes to diagnose who contributed volume changes.

## Typical Use Cases

- Capacity planning (human vs AI load share)
- Incident analysis (drop in throughput)
- Team productivity reporting
